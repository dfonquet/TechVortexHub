CCIE-SP EVE-NG Lab Topology
Author: Daniel Fonque

This artifact contains the EVE-NG topology export used as part of my CCIE Service Provider lab practice.

Included:
- EVE-NG lab export package
- CCIE-SP-LB.unl topology file inside the EVE-NG export

Not included:
- Cisco IOS, IOS XE, IOS XR, Juniper, or any other vendor software images
- Licensed router/switch images

Notes:
- This lab is shared for study and documentation purposes.
- You need your own legally obtained network images to import and run the topology.
- The lab is connected to my public CCIE-SP notes covering IS-IS, SR-MPLS, MP-BGP, L3VPN, NAT, multicast, RR design, and operational validation.
